# 📰 NewsMania

NewsMania is a **full-stack news and community web application** where users can share posts, interact with others, and stay updated with trending discussions.  
It supports authentication, community posting, likes, comments, and real-time interaction.

---

## 🚀 Features

- 🔐 User Authentication (Signup / Login)
- 📰 Create and share posts
- 👍 Like posts
- 💬 Comment system
- 👤 User profiles
- ⚡ Real-time updates using Socket.IO
- 📱 Responsive UI
- 🗄 MongoDB database

---
